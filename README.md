# Multerm

Multerm is a cross-platform SSH client built with C# and Avalonia. It provides a modern GUI to manage multiple SSH connections, store encrypted SSH profiles, and simplify SSH session management. Multerm supports plugins for extending functionality and is designed to be secure, customizable, and collaborative.

## Features 
- Cross-platform support (Windows, macOS, Linux)
- Manage multiple SSH connections with a tabbed interface
- Encrypted SSH profile storage for secure credential management (planned for future releases)
- Customizable terminal themes and fonts (planned for future releases)
- Plugin system for extending functionality (planned for future releases)

## Installation
1. Download the latest official release from the [GitHub releases page](#)
2. Extract the downloaded archive
3. Run the executable file to start Multerm

> Note: You may also compile the source for personal use.
